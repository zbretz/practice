This is an imaginging of the applicant profile. On this page you'll find recorded interviews conducted with a live, trained interviewer experienced in hiring engineers.

Click on an interview box to pop-out a modal with the interview and timestamps of critical (informative, revealing, etc) moments and comments (feedback, alysis, review) from the interviewer.

In the sidebar to the left, you can find links to other components of the interview process. Coding challenges provides the prompts to toy problems and the solutions in an interactive IDE for inspecting candidate's code and running it through an interpreter. The Coding Project currently links to an external site that allows you to run React code in a sandboxed-environment. This would be both what the applicant sees (and writes in) and what the examiners (and hiring companies) can access for inspection and evaluation. Perhaps our implemetation will utilize a similar sandbox experience.
